# A Conservative and Positivity-Preserving Baseline Scheme for Shallow-Water Flow with Suspended Sediment Deposition

Author: Isaac Boateng Numoah, Old Dominion University

This is Preprint v1 and is not yet peer reviewed.

This repository contains the compiled PDF, XeLaTeX source, and three required diagnostic figures.

The work proves conservation and positivity properties only under its stated restricted periodic, flat-bed assumptions.

It makes no real-river validation or broad novelty claim.

Compile with XeLaTeX using `main.tex`.
